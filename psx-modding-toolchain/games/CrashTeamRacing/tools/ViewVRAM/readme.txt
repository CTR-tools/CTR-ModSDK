       PS显存数据查看工具
             v 2007.2.8
                    Agemo

这个工具是什么：
　查看PS显存数据的工具。
　支持PS所有的显示模式(各种模式参照文档 GPU.txt)
　支持1MB的显存dump文件(PCSX DUMP)，以及epsxe，adripsx的即时存档。

系统需求： 
　分辨率1024x768以上，16位色以上。
　支持 98/2000/xp

使用方法：
　1.  如果是比如epsxe模拟器的存档。存档经过zip算法压缩过，所以先把
　　　存档文件后缀改为.zip然后解压，得到4mb左右的原始数据。
　    把解压后文件拖到本软件界面上就可以显示了
  2.  对于pcsx dump. 软件默认读入当前目录下面的vram.bin

显示画面的解释：
　蓝线以上是 1024x512 的原始显存图，以16bit模式显示。
　蓝线以下是红色的框范围（以后叫窗口）以内在不同模式下的显示。
　所有的提示信息都显示在窗口标题。

键控制
・鼠标左键或者按回车：重新读入显存文件
・w,s,a,d 移动窗口
・1,2,3,4,8,9,0 切换显示模式，具体显示模式见下:
　　1 = 4bit grey（灰度图，即忽略CLUT）
　　2 = 8bit grey 
　　3 = 16bit (和蓝线上是一样的模式)
　　4 = 24bit 
　　8 = 24bit MDEC (就是动画)
　　9 = 4bit clut (与grey模式区别是用了clut)
　　0 = 8bit clut 
・~ 反色，对于4bit色的反色模式，是指按双层TIM字库进行反色
・方向箭头 CLUT起始坐标（屏幕上看到的红色十字），仅用于9,0的clut模式。
・SHIFT 微调。按w,s,a,d和方向箭头一下改变16象素, 如果按住shift, 改变1
・CTL + 鼠标移动 观看坐标（十进制），以及在显存IMB的偏移量（十六进制）


注1、使用此工具前最好对psx各种显示格式有了解。gpu.txt
   　http://psx.rules.org/

注2：寻找CLUT的技巧
     CLUT非常明显，就是一条彩线。
     8bit的clut长度256。4bit的clut长度16
     CLUT在显存中横坐标必须是16的整数倍（这也是为什么
     设定按方向键左右一下是16象素的原因）

注3：epsxe 显存开始于存档进度解压后偏移 $2733DF 处。(1.52和1.60如此) 
   　adripsx 显存开始于存档进度偏移 0，没压缩。
   　程序自动识别以上两种
   　同时支持 p.e.o.p.s. soft 显示插件的抓图也是显存，不过插件在
     16位色变换为24位色BMP时候失真了，所以没有实际意义。
